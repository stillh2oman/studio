# **App Name**: Designer's Ledger

## Core Features:

- Project and Client Management: Establish and manage a database of projects and clients, including client contact details (name, email). Provide options to create new projects or clients as needed.
- Billable Hours Tracking: Record billable hours for projects, including client information, work description, hours worked, and rate of pay, with an automatic total calculation. Integrate multi-select invoice status options ('Not Sent', 'Invoice Sent', 'Past Due', 'Paid') and single-select discounts.
- Print Job Tracking: Track print jobs with details such as project and client names, paper size, rate per sheet, and number of sheets, calculating a total cost. Incorporate multi-select invoice status options identical to billable hours.
- Integrated Data Lookup: Effortlessly retrieve and associate existing project and client data for both billable hours and print jobs, ensuring client email addresses are automatically linked to client names.
- Persistent Data Storage: Securely store all project, client, billable hour, and print job data in a robust database, allowing for consistent data access and retrieval across sessions.
- Comprehensive Invoice Status Reporting: Maintain and visualize the invoice status for all billable hours and print jobs, enabling easy identification of tasks needing invoicing or follow-up.
- AI Work Description Assistant: An AI tool that suggests professional and relevant descriptions for billable work entries based on user input keywords or learned patterns from previous descriptions, enhancing efficiency.

## Style Guidelines:

- Dark color scheme emphasizing a sophisticated and professional aesthetic.
- Background color: Deep charcoal blue (#1F2A2E), providing a calm and focused canvas reminiscent of the logo's dark background.
- Primary interactive color: Rich burgundy red (#8E2431), drawing from the 'INK' part of the logo to highlight active elements and important information with professional vibrancy.
- Accent color: Vibrant coral orange (#FAA172), offering a warm and inviting contrast for calls to action or notifications, standing out against the dark scheme.
- Headline font: 'Alegreya' (humanist serif) for elegant and intellectual titles, reflecting the 'designer' aspect.
- Body font: 'Inter' (grotesque sans-serif) for clear and objective readability in data tables and forms, ensuring efficiency.
- Minimalistic line icons that align with the app's clean and professional visual style, maintaining clarity and modern aesthetics.
- Intuitive tabbed interface separating 'Billable Hours' and 'Printing' sections, featuring organized data tables and user-friendly input forms for streamlined workflow.
- Subtle and purposeful animations for user feedback upon data entry, saving actions, or status updates, contributing to a fluid and responsive experience.
