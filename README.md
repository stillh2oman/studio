
# Designer's Ink Command Center

Professional Command Center for tracking billable hours, print jobs, and architectural design tasks by Jeff Dillon.

## Key Features

- **Unified Billing**: Combined module for Billable Hours and Print Jobs.
- **Architectural Pipeline**: Real-time project status tracking and checklist management.
- **Precision Checklist**: Detailed 10-document verification system with 60+ technical sub-tasks.
- **Project Notes**: Collaborative activity logs with voice-to-text integration.
- **Staff Command**: Granular permission management and secure database for PII.
- **Time Blocking**: Integrated calendar for scheduling focus blocks and client meetings.
- **Plan Database**: Firm-wide blueprint registry and library.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Database/Auth**: Firebase Firestore & Firebase Authentication
- **Styling**: Tailwind CSS & ShadCN UI
